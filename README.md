### BRAZUCA

Mobile App using World Cup 2014 Kimono Labs API, Ionic Framework and Cordova.


```bash
$ npm install -g ionic cordova harp newman testem phantomjs casperjs
```

### Development

```bash
$ harp compile app www
```
